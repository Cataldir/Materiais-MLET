# 03 — Pipeline ML

> 8 aulas · ~360min de video


## Resumo da Disciplina

Esta disciplina cobre 8 temas progressivos: desde fundamentos até aplicação em produção. Os primeiros temas abordam: Introdução ao Pipeline de ML; Ingestão de Dados e Feature Engineering; Treinamento de Modelos e Validação; Deploy de Modelos (Implantação Inicial).
