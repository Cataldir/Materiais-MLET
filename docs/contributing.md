# Contributing Notes

This repository is a student reference surface. Contributions should improve the clarity, accuracy, or placement of teaching material without adding private operations context or automation scaffolding.

## Placement Rules

1. Put reusable teaching material under the matching phase, discipline, and lesson path.
2. Put live-session material under the discipline it supports, using a local `lives/` folder.
3. Put study-group material under `fase-*/grupos-de-estudo/`.
4. Keep repository navigation, coverage notes, and editorial history under `docs/`.
5. Keep agent definitions, workflow automation, test harnesses, and repository-maintenance tools outside this student-facing repository.

## Content Rules

1. Do not publish private scheduling, professor follow-up, email evidence, tickets, grades, or student identifiers.
2. Do not promote caches, checkpoints, experiment traces, model binaries, or generated training artifacts unless they are intentionally curated as small teaching examples.
3. Prefer the nearest phase, discipline, or lesson README over parallel documentation.
4. Keep cohort labels as provenance for dated sessions, not as the primary navigation model.
