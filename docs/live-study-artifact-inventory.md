## Achados de avaliação

- Todos os materiais de live e grupo de estudo possuem README local.
- As lives técnicas simples geralmente já tinham `exercicio.py`, mas faltavam roteiro explícito, atividade do aluno e checklist de engenharia.
- As lives de grupos de estudo não possuem código local; isso é adequado porque o objetivo é acompanhamento, reflexão e conexão com Tech Challenge.
- As referências variam em profundidade. Onde o README era mínimo, os novos artefatos adicionam referências por fase e orientam o estudante a consultar o README local.
- Nenhum artefato novo cria resposta final, teste global, ferramenta de manutenção ou atalho de validação na raiz.
