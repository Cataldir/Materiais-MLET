# Fase 05 — Deploy Avançado de IA Generativa

## Tech Challenge: Datathon — Caso Real com Empresa Convidada

O Tech Challenge da Fase 05 assume o formato de **Datathon**: uma competição técnica baseada em um problema real fornecido por uma empresa convidada. O Datathon funciona como projeto integrador — os grupos devem mobilizar todas as competências construídas ao longo das Fases 01 a 05 para entregar uma solução completa de ML Engineering.

---

## Entrega do Tech Challenge

- **Obrigatório:** Repositório GitHub + Pitch no Demo Day (≤ 10 min + 5 min Q&A)
- **Opcional:** Deploy em ambiente de produção em nuvem

## Bibliotecas Requeridas

- **PyTorch** · **Scikit-Learn** · **MLflow** · **PyTorch Lightning**
- **LangChain ou LlamaIndex** · **vLLM ou BentoML** · **RAGAS** · **Langfuse ou TruLens**

## Dinâmica com Empresa

```
T-30 dias → Convite à empresa
T-15 dias → Empresa entrega briefing + dados + critérios de sucesso
T-0       → Kickoff com apresentação da empresa
T+2 sem   → Check-in intermediário Q&A
Final     → Demo Day (Pitches) — Empresa + Banca avaliam
```
