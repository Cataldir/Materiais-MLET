# 📖 Grupos de Estudo — MLET Pós-Tech

> Materiais de apoio para os Grupos de Estudo (GE) organizados por **fase** e alinhados às **etapas do Tech Challenge**.

---

## Estrutura

Cada fase contém **4 Grupos de Estudo**, cada um mapeado diretamente para uma etapa do Tech Challenge da fase correspondente. O objetivo é preparar os alunos progressivamente para a entrega final do TC.

| Fase | Tema | Tech Challenge |
|------|------|----------------|
| [Fase 01](fase-01-produtizacao-de-modelos/) | Produtização de Modelos | Rede Neural para Previsão de Churn com Pipeline End-to-End |
| [Fase 02](fase-02-containers-e-ambientes-reprodutiveis/) | Containers e Ambientes Reprodutíveis | Pipeline de Treinamento Containerizado, Versionado e Reprodutível |
| [Fase 03](fase-03-cloud-e-mlops/) | Cloud e MLOps | Deploy de Rede Neural em Produção com CI/CD, Monitoramento e Otimização |
| [Fase 04](fase-04-monitoramento-e-governanca/) | Monitoramento e Governança | Monitoramento End-to-End com Drift, Observabilidade e Governança |
| [Fase 05](fase-05-deploy-avancado-de-ia-generativa/) | Deploy Avançado de IA Generativa | Datathon — Caso Real com Empresa Convidada |

---

## Formato das Sessões

Cada Grupo de Estudos segue o roteiro padrão:

1. **Revisão conceitual** (~20 min) — Recapitular pontos centrais da disciplina
2. **Exercício guiado** (~40 min) — Resolver problema prático em grupo
3. **Discussão aberta** (~20 min) — Dúvidas, aplicações e próximos passos
4. **Conexão com Tech Challenge** (~10 min) — Como aplicar no projeto da fase

---

## Como Usar

- Antes do GE: revise o material da disciplina de referência
- Durante o GE: siga o roteiro e os exercícios propostos
- Após o GE: aplique os conceitos na etapa correspondente do Tech Challenge
