## Requisitos de Hardware

- Aulas de deploy de LLMs (01, 02): GPU recomendada (NVIDIA com ≥8GB VRAM)
- Aulas de agentes (02, 03): CPU suficiente com modelos via API
- Aulas de avaliação (04): CPU com acesso à internet para APIs
