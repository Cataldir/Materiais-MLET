# Fase 01 — Produtização de Modelos

## Tech Challenge: Rede Neural para Previsão de Churn com Pipeline Profissional End-to-End

Uma operadora de telecomunicações precisa de um modelo preditivo de churn que classifique clientes com risco de cancelamento. O grupo deve construir o projeto do zero ao modelo servido via API, aplicando todas as boas práticas de engenharia de ML.

## Entrega do Tech Challenge

- **Obrigatório:** Repositório GitHub + Vídeo de 5 minutos (método STAR)
- **Opcional:** Deploy em ambiente de produção em nuvem (AWS, Azure ou GCP)

## Bibliotecas Requeridas

- **PyTorch** — Construção e treinamento da rede neural (MLP)
- **Scikit-Learn** — Pipelines de pré-processamento e modelos baseline
- **MLflow** — Tracking de experimentos
- **FastAPI** — API de inferência do modelo
