# Referenciais teoricos da disciplina

Este indice organiza as referencias recorrentes de apoio da disciplina. Registre
aqui leituras, documentacao e materiais de contexto quando forem relevantes
para aula, grupo de estudo, live ou Tech Challenge.

## Base

- Sato, D. *Continuous Delivery for Machine Learning*. O'Reilly, 2023.
  ISBN 978-1-098-12661-5. Capitulos sobre data validation e testing.
- Huyen, C. *Designing Machine Learning Systems*. O'Reilly, 2022.
  ISBN 978-1-098-10796-6. Capitulo 8: Data Distribution Shifts.
- Breck, E. et al. "The ML Test Score: A Rubric for ML Production Readiness
  and Technical Debt Reduction." *IEEE Big Data*, 2017.
  DOI: 10.1109/BigData.2017.8258038.

## Complementar

- Schelter, S. et al. "Automating Large-Scale Data Quality Verification."
  *Proceedings of the VLDB Endowment*, vol. 11, no. 12, 2018.
  DOI: 10.14778/3229863.3229867.
- Polyzotis, N. et al. "Data Lifecycle Challenges in Production Machine Learning."
  *ACM SIGMOD Record*, vol. 47, no. 2, 2018.
- Sculley, D. et al. "Hidden Technical Debt in Machine Learning Systems."
  *NeurIPS*, 2015.
